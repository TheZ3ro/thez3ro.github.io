Features

Easy Address Managment

You can add your address simply with QR Code intead of copying it by hands.

Multi Address Support

You can manage multiple address even with the same altcoin.

Real-Time USD-Altcoin Price

The application display the real-time exchange price from MtGox.com and Bter.com .

What is an Anti-Wallet?

It's an Application or a Program where you can only see your balance (from blockexplorer) without having all the transaction history and without a new wallet, but you can't make any transaction.
You can see your wallet balance without privatekey or a true wallet.

Much Altcoin, Such Power

BitZero can manage many altcoin with no problem.

Easy, Secure and Free

BitZero is very easy to use, once you install it you have to add one address and then it's ready-to-use.
It's very secure, it's 100% free of malware and scam and comes with Bitcoin's security.
It's free-software, you can download the source code and edit it freely.
